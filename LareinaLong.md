# 2. 学习了解区块链基础概念
### 一、什么是区块链 & 核心特性
区块链是一种去中心化的分布式账本，由多个节点共同维护、不依赖中心权威，保证数据不可篡改与公开透明。
每个区块包含交易数据与上一块的哈希，串联形成链条；任意篡改都会破坏后续链的完整性。
### 二、比特币机制与代币激励
节点（矿工）通过验证交易、打包区块获得加密代币奖励，这是驱动区块链网络运行的经济机制。
比特币具有稀缺性、匿名性、透明性等属性，但因其网络设计限制确认速度较慢，市场价格波动较大。
### 三、公链、联盟链与私链对比
- 公链：完全开放、任何人可参与，透明度高但效率相对较低，典型应用：比特币、以太坊。
- 联盟链：由预先筛选的组织组成联盟，共同治理，适合企业合作场景如供应链、金融协作。
- 私链：由单一组织控制访问与管理规则，集中权限，高效率高隐私，但缺乏去中心化特征。
### 四、Web3 与 Web2 / Web 3.0 的差异
- Web2：中心化平台主导，用户非数据拥有者（如微信、抖音）。
- Web 3.0（语义网）：强调机器可读数据与语义结构化，不依赖区块链技术。
- Web3：基于区块链，用户拥有身份和资产，智能合约自动执行规则，支持 DeFi、NFT、DAO 等去中心化应用。
